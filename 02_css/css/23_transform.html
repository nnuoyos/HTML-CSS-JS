transform : 요소를 여러가지 방법으로 변형

  1. translate : 위치이동
    translateX /transformY
  2. scale : 크기변경
    글자랑 사진도 같이 억지로 구겨서 바꾸기 때문에 잘 사용하지 않음 scaleX scaleY
  3. rotate : 회전
    rotateX rotateY
  4. skew : 뒤틀기
    잘 사용하지 않음
    skewX skewY


transform : translate(x,y);
transform : scale(가로N배, 세로N배);
transform : rotate(각도deg);
transform : skew(각도deg, 각도deg);
