transition : 시작에서 종료까지의 변화를 시각화

transition : all 0s;(property duration)

속기법 transition : property duration timing-function;(전부 타이핑하면 이렇게 길다)

트랜지션은 마우스를 올리고 내릴 때 시각화 되어야 하므로 거의 :hover랑 같이 써야한다
:hover전후의 사용으로 효과의 차이가 있다

**마우스를 올리고 내릴 때 모두 시각화 하고싶다면 :hover 주기 전 상태에서 트랜지션 적용**

  1. transition-property : 트랜지션을 적용할 css속성 이름;
  2. transition-duration : 트랜지션이 일어나는 지속시간(0s);
  3. transition-timing-function : 속성의 중간값을 계산;
  4. transition-delay : 대기시간, 속성이 변한 시점과 트랜지션이 실제로 시작하는 사이에 기다리는 시간(0s);
